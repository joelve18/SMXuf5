Lista de definiciones sobre HTML y CSS

HTML
HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Define la estructura y el contenido de una página mediante etiquetas y elementos.
CSS
CSS (Cascading Style Sheets) es un lenguaje utilizado para definir la presentación y el diseño de las páginas web escritas en HTML. Permite estilizar y dar formato al contenido HTML.
Etiqueta
En HTML, una etiqueta es un elemento utilizado para definir diferentes partes de un documento web. Las etiquetas están rodeadas por corchetes angulares y pueden contener atributos y contenido.
Atributo
En HTML, un atributo proporciona información adicional sobre un elemento HTML y se especifica dentro de la etiqueta de apertura. Los atributos pueden modificar el comportamiento o la apariencia de los elementos.
Selector
En CSS, un selector es un patrón utilizado para seleccionar el elemento HTML al que se aplicarán los estilos. Los selectores pueden ser etiquetas HTML, clases, IDs, entre otros.
